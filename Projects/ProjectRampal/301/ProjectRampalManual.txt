ProjectRampal.exe manual

Call the program with either no or two parameters:

ProjectRampal.exe 
ProjectRampal.exe [theta] [phylogeny]

For example:

ProjectRampal.exe 
ProjectRampal.exe 10.0 ((2,2),2)

If the program is called without parameters, the parameters are read from the files 'theta.txt' and 'phylogeny.txt'.

The result will be written to screen and to the file 'Results.txt'
