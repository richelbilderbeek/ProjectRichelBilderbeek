(C++) std::random_shuffle

 

std::random_shuffle is an STL algorithm to shuffle all elements in a container.

 

 

 

 

 

Project and source code

 

Operating system: Ubuntu 10.04 LTS Lucid Lynx

IDE: Qt Creator 2.0.0

Project type: Qt4 Console Application

Compiler: G++ 4.4.1

Libraries used:

  • Boost: version 1.40
  • STL: from GCC, shipped with Qt Creator 2.0.0

 

  • Download the Qt Creator project 'CppRandom_shuffle' (zip)

 

 

 

 

 

main.cpp

 

#include <algorithm>
#include <iostream>
#include <vector>
#include <boost/foreach.hpp>

int main()
{
  std::vector<int> v;
  for (int i=0; i!=5; ++i) v.push_back(i);

  std::cout << "Before shuffling:\n";

  BOOST_FOREACH(int i, v) std::cout << i << '\n';

  std::random_shuffle(v.begin(),v.end());

  std::cout << "After shuffling:\n";

  BOOST_FOREACH(int i, v) std::cout << i << '\n';
}
