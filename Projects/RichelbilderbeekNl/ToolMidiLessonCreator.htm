(C++) MidiLessonCreator

 

MidiLessonCreator is a tool to create MIDI lessons.

Technical facts

 

Application type(s)

  • Desktop application

Operating system(s) or programming environment(s)

  • Lubuntu 14.10 (utopic)

IDE(s):

  • Qt Creator 3.1.1

Project type:

  • GUI application

C++ standard:

  • C++11

Compiler(s):

  • G++ 4.9.1

Libraries used:

  • Qt: version 5.3.0 (32 bit)
  • STL: GNU ISO C++ Library, version 4.9.1
