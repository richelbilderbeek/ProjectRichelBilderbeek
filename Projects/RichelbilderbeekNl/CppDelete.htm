(C++) delete

 

delete is a keyword to release dynamically allocated memory (allocated by the keyword new).

 

 

 

 

 

Examples

 

  • delete example 1: basic

 

 

 

 

 

Advice

 

  • Avoid 'naked' new and delete [2,3]
  • Use smart pointers to avoid forgetting to delete objects created using new [1,4]

 

Be aware that it is valid to delete a pointer-to-const.
