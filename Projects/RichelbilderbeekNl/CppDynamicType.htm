(C++) dynamic type

 

  • dynamic type <glvalue> type of the most derived object to which the glvalue denoted by a glvalue expression refers. Example: if a pointer p whose static type is 'pointer to class B' is pointing to an object of class D, derived from B, the dynamic type of the expression *p is 'D'. References are treated similarly [1]
  • dynamic type <prvalue> static type of the prvalue expression [2]
