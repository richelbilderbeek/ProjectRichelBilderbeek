(Tool) Wine

 

Wine is a tool to (not) emulate a Windows environment under Linux.

 

Among others, Wine can be used to (in some definitions) port programs from Linux to Windows, because it allows to run the Windows executable on the Linux platform.

 

To create native Windows executables without having a computer with Windows installed on, Wine can be used to install Cygwin, which in turn can create Windows executables from its UNIX-like environment.

 

The answer to the Qt FAQ Can Qt Creator run under Wine? seems to be no, however.
