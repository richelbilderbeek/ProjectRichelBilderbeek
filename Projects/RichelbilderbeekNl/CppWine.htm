(C++) Wine

 

Wine is a system environent (which in my definition is sort of an operating system), as it (does not) emulates a Windows environment under Linux, to run Windows under Wine under Ubuntu. Among others, Wine can be used to (in some definitions) port programs from Linux to Windows.

 

The answer to the Qt FAQ Can Qt Creator run under Wine under Ubuntu? seems to be no, however.
