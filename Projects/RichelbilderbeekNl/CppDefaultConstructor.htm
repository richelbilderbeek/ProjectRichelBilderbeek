(C++) default constructor

 

A default constructor is a constructor that has no arguments or only default arguments.

 

struct MyClass
{
  MyClass() {}; //A default constructor
  //Rest of the class
};

 

struct MyClass
{
  MyClass(int x) : m_x(x) {}; //A default constructor
  //Rest of the public part of the class
  private:
  int m_x;
  //Rest of the private part of the class
};

 

 

 

 

 

Advice

 

  • Give a class a default constructor if and only if there is a "natural" default value [1]
