(C++) std::showpoint

 

std::showpoint is an STL stream manipulator to show the zeroes behind the comma of a double.

 

  • Download the Qt Creator project 'CppShowpoint' (zip)

 

#include <iostream>
#include <iomanip>
int main ()
{
  std::cout
    << std::setprecision(10)
    << std::noshowpoint << 1.0 << '\n'
    << std::showpoint   << 1.0 << '\n';
}

 

Screen output:

 

1
1.000000000
