(C++) std::strtod

 

std::strtod is an STL function to convert a std::string to double.

 

 

 

 

 

Example: ToDouble

 

#include <cstdlib>
#include <string>

//From http://www.richelbilderbeek.nl/CppToDouble.htm
double ToDouble(const std::string& s)
{
  return std::strtod(s.c_str(),0);
}
