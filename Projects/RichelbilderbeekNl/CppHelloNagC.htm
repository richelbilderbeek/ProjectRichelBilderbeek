(C++) Hello NAG using C

 

Hello NAG using C is a Hello World program to test the NAG library.

 

  • Download the Qt Creator project 'CppHelloNagC' (zip)

 

 

 

 

 

Technical facts

 

Application type(s)

  • Desktop application

Operating system(s) or programming environment(s)

  • Lubuntu 12.10 (quantal)

IDE(s):

  • Qt Creator 2.5.2

Project type:

  • Console application

C++ standard:

  • C: How to use Qt Creator to compile C code?

Compiler(s):

  • G++ 4.7.2

Libraries used:

  • STL: GNU ISO C++ Library, version 4.7.2

 

 

 

 

 

Qt project file: CppHelloNagC.pro

 

QT       -= core
QT       -= gui

INCLUDEPATH += /opt/NAG/cll3a09dgl/include

LIBS += -L/opt/NAG/cll3a09dgl/lib -lnagc_nag

#opt/NAG/cll3a09dgl/lib/libnagc_nag.so:-1: error: undefined reference to `sqrt'
QMAKE_LFLAGS += -lm
LIBS += -lm

#Compile with GNU C compiler
QMAKE_CXX = gcc
QMAKE_CXXFLAGS = -x c

CONFIG   += console
CONFIG   -= app_bundle
TEMPLATE = app
SOURCES += main.cpp

 

 

 

 

 

main.cpp

 

#include <math.h>
#include <nag.h>
#include <nag_stdlib.h>

int main(void)
{
  char * s = 0;
  s = NAG_ALLOC(11,char);
  if (!s)
  {
    printf("Cannot allocate memory for s");
    return 1;
  }
  sprintf(s, "Hello NAG");
  puts(s);
  NAG_FREE(s);
  return 0;
}
