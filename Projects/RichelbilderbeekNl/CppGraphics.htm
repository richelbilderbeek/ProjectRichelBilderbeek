Graphics code snippets are code snippets for doing graphics.

 

Most VCL graphics code snippets can be found at the VCL graphics code snippets page.

 

 

 

 

 

  1. AddColoredNoise : add colored noise
  2. ASCII art : perform ASCII art
  3. Bleach : bleach an image (that is: make it whiter)
  4. ColorToRgb : convert a color struct to its RGB values
  5. ConvertToGrey : convert a colored image to a grey image
  6. Darken : darkens an image (that is: make it blacker
  7. DoFilterOperation : perform a filter operation ('convolution') on an image
  8. DoHistogramEqualization : perform a histogram equalization
  9. DrawGlobe : draw a globe
  10. DrawGlobeMinus : draw a globe with a minus on it
  11. DrawGlobePlus : draw a globe with a plus on it
  12. DrawLymphocyte : draw a lymphocyte
  13. DrawTextCentered : draw text centered on image
  14. EnhanceContrast : enhances the contrast of a grey image
  15. GetAverageGreyness: calculate the average greyness of an image
  16. GetFontHeight : get a font's height
  17. GetGreyness : get the greyness of one or more pixels
  18. GetImageHistogram : get a histogram of an image's grey intensities
  19. GetPixel : get the color of one or more pixels
  20. MakeScreenshot : make a screenshot and store it as a bitmap
  21. Paint : paint an image to a single color fast
  22. Pixelate : pixelate an image (that is: make it blockier)
  23. Rainbow : obtain a rainbow-like color gradient
  24. RemoveColor : remove a color
  25. RgbGradient : obtain an RGB gradient
  26. RgbToColor: convert RGB values to color struct
  27. Rotate : rotate an image
  28. SetGreyness : set a pixel's greyness
  29. SetPixel : set a pixel's color
  30. SurfacePlotterTImage : draw a surfacePlot
  31. ThresholdFilter : perform a threshold filter operation
