(C++) std::right

 

std::right is a stream manipulator to let the minus and the digits of a negative number both be put at the right of the padding width.

 

#include <iomanip>
#include <iostream>

int main ()
{
  const int x = -12;
  std::cout
    << std::setw(4)  << std::internal << x << '\n'
    << std::setw(4)  << std::left     << x << '\n'
    << std::setw(4)  << std::right    << x << '\n';
}

 

Screen output:

 

- 12
-12 
 -12
