(C++) MxeExample4

 

 

MXE example 4: Hello Boost is an MXE example to cross-compile a Hello Boost program from GNU/Linux to Windows.

 

  • Download the Qt Creator project 'CppMxeExample4' (zip)

Technical facts

 

Operating system(s) or programming environment(s)

  • Lubuntu 14.10 (utopic)

IDE(s):

  • Qt Creator 3.1.1

Project type:

  • Console application

C++ standard:

  • C++98

Compiler(s):

  • G++ 4.9.1

Libraries used:

  • STL: GNU ISO C++ Library, version 4.9.1

 

 

 

 

 

Qt project file: ./CppMxeExample4/CppMxeExample4.pro

 

QT       -= core
QT       -= gui
CONFIG   += console
CONFIG   -= app_bundle
TEMPLATE = app
SOURCES += main.cpp

 

 

 

 

 

./CppMxeExample4/main.cpp

 

#include <iostream>
#include <boost/lexical_cast.hpp>

int main()
{
  if (boost::lexical_cast<std::string>(123) == "123")
  {
    std::cout << "Hello Boost" << '\n';
  }
}
