(Command line) Get the C++ Standard draft

 

Get the C++ Standard draft is a command line snippet to obtain the C++ Standard draft.

 

 

git clone https://github.com/cplusplus/draft.git
sudo apt-get install latexmk
cd draft
latexmk -pdf std
