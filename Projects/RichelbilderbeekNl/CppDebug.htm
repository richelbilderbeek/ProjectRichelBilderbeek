(C++) Debug

 

Debugging is the process of removing bugs.

 

Tools for detecting memory management and threading bugs are, among others, valgrind (Linux) and DrMemory (Windows).

 

The process of improving the runtime speed is called profiling.

 

 

 

 

Advice

 

  • The key question in debugging is: How would I know if the program actually worked correctly? [1]
  • When defining a template, first design and debug a non-template version; later generalize by adding parameters [2]
  • Making the data members of a class private and the member functions of the class public facilitates debugging [3]
