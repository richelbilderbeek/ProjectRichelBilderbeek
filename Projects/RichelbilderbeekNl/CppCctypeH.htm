(C++) cctype.h

 

cctype.h is the STL header file for determining character types like lowercase and uppercase.

 

cctype.h contains the definitions of (incomplete list):

  • std::isalnum
  • std::isalpha
  • std::iscntrl
  • std::isdigit
  • std::isgraph
  • std::islower
  • std::isprint
  • std::ispunct
  • std::isspace
  • std::isupper
  • std::isxdigit
  • std::tolower
  • std::toupper

 

 

 

 

 

#include <cctype>
