(C++) nullptr

 

nullptr is a keyword that has different meanings, depending on the standard used:

  • nullptr in the C++98 standard
  • nullptr in the C++11 standard

 

 

 

 

 

nullptr in the C++98 standard

 

nullptr is not supported in C++98.

 

 

 

 

 

nullptr in the C++11 standard

 

  • Download the Qt Creator project 'CppNullptr' (zip)

 

nullptr is a keyword to indicate an unitialized pointer.

 

#include <cassert>

int main()
{
  //Create a new p
  int * p = new int(3);
  assert(*p == 3);

  //Get rid of the current p
  delete p;
  p = nullptr;

  //Create a new p
  p = new int(4);
  assert(*p == 4);
}
