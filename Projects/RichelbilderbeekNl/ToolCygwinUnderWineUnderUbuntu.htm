(Tool) Running Cygwin under Wine under Ubuntu

 

Cygwin does not appear to run well under Wine under Ubuntu

 

To run Cygwin under Wine, follow [1]:

  • Navigate to ~/.wine/drive_c/cygwin
  • wineconsole cmd
  • Cygwin.bat

 

For me, this resulted in

  • the Cygwin shell (png)
  • the wineconsole shell (png)

 

Note the error (in the Cygwin shell):

 

CMD Version 1.2

C:\cygwin>Cygwin
bash: /dev/null: No such file or directory
bash: cannot create temp file for here document: Invalid request code
Your group is currently "passwd/group_GID_clash(gid/pgsid)".  This
indicates that your gid is not in /etc/group, but the pgsid (primary
group associated with your SID) is in /etc/group.

The /etc/passwd (and possibly /etc/group) files should be rebuilt.
See the man pages for mkpasswd and mkgroup then, for example, run

mkpasswd -l [-d] > /etc/passwd
mkgroup  -l [-d] > /etc/group

Note that the -d switch is necessary for domain users.
