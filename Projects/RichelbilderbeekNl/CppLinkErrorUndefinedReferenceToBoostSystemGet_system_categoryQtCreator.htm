Source code

 

If you want to work with Boost.Asio:

 

#include <boost/asio.hpp>

 

Or if you want to work with Boost.Filesystem:

 

#include <boost/filesystem.hpp>

 

 

 

 

 

Solution

 

Add the following line to the Qt project file:

 

LIBS += -lboost_system
