(C++) std::call_once

 

std::call_once is a C++11 standard function to call another function exactly one time.

 

  • Download the Qt Creator project 'CppCall_once' (zip)

  • C++11

  • G++ 4.6.1

  • STL: GNU ISO C++ Library, version 4.6.1

main.cpp

 

#include <iostream>
#include <thread>

void f() { std::cout << "Hello world\n"; }

int main()
{
  std::once_flag flag;
  std::call_once(flag,&f);
  std::call_once(flag,&f);
  std::call_once(flag,&f);
}
