(Music) ABC notation

 

This is my ABC notation page, mostly for my own reference really...

 

 

 

 

 

From an abc text file to something else

 

What?

Ubuntu terminal command

Output filename

Convert ABC to Midi

abc2midi filename.abc

filename1.mid

Convert ABC to postscript

abcm2ps filename.abc

Out.ps

Play the Midi file

playsound filename1.mid

(none)

Transpose ABC file

abc2abc filename.abc -t 1 > filename2.abc

filename2.abc
