Job offerings

 

I am open to job offerings, for the following functions:

  • Full-time PhD in Theoretical Biology or related disciplines
  • Full-time free software C++ developer
  • Four-days-a-week mechatronics intern

 

More details below.

 

 

 

 

PhD

 

For a PhD position, I require:

 

  • An interesting problem (which should be easy)
  • A supervisor that has synergy with me
  • I am willing to move, also abroad
  • Willing to work from home or commute 90 minutes per day by public transportation
  • My source code must be open source
  • Wages at least Dutch minimum wages, which is about 1500 euro (2000 USD) gross per month. Transportation costs, if any, should be paid additionally

 

 

 

 

 

Free software C++ developer

 

For a free software C++ developer position, I require:

 

  • A competent supervisor with high moral standards
  • Willing to work from home or commute 90 minutes per day by public transportation
  • Wages at least Dutch minimum wages, which is about 1500 euro (2000 USD) gross per month. Transportation costs, if any, should be paid additionally

 

Note that I will not write non-free software, for any price.

 

As a developer, I would hire myself if:

  • I needed a defensive programmer, because the software needs to be correct and thus thoroughly tested
  • I needed to convert a C++ desktop application to an interactive website
  • I needed to have a desktop application for the most important platforms
  • I needed to have a literate programmer, that knows the experts' advice, for cleanly designed and written code

 

 

 

 

 

Mechatronics

 

For a Mechatronics intern position, I require:

 

  • A competent supervisor with high moral standards
  • Willing to work from home or commute 90 minutes per day by public transportation
  • The need to go to school on Wednesdays
  • Wages at least Dutch minimum wages, which is about 1200 euro (1650 USD) gross per month (as it is 0.8 FTU) Transportation costs should be paid additionally. School should be paid (but appears to be fully tax-deductible)
