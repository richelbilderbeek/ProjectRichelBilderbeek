(C++) std::wclog

 

std::wclog is the standard output stream similar to std::clog, yet for wide characters.

 

  • Download the Qt Creator project 'CppWclog' (zip)

 

 

 

 

 

Technical facts

 

Application type(s)

  • Desktop application

Operating system(s) or programming environment(s)

  • Lubuntu 12.10 (quantal)

IDE(s):

  • Qt Creator 2.5.2

Project type:

  • Console application

C++ standard:

  • C++11

Compiler(s):

  • G++ 4.7.2

Libraries used:

  • STL: GNU ISO C++ Library, version 4.7.2

 

 

 

 

 

Qt project file: CppWclog.pro

 

TEMPLATE = app
CONFIG += console
CONFIG -= qt
QMAKE_CXXFLAGS += -std=c++11 -Wall -Wextra -Weffc++ -Wall
SOURCES += main.cpp

 

 

 

 

 

main.cpp

 

#include <iostream>

int main()
{
  std::cout  << "Hello World from std::cout\n";
  std::clog  << "Hello World from std::clog\n";
  std::cerr  << "Hello World from std::cerr\n";
  std::wcout << "Hello World from std::wcout\n";
  std::wclog << "Hello World from std::wclog\n";
  std::wcerr << "Hello World from std::wcerr\n";
}
