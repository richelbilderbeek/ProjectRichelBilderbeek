(C++) C++ Builder

 

C++ Builder is a Windows-only C++ IDE that uses the VCL library for GUI development.

 

Because I have moved to the cross-platform Qt Creator IDE, I will not work a lot on the C++ Builder pages anymore.

 

Nowadays C++ Builder is called 'Borland Developer Studio', 'Turbo C++ Explorer' or RAD Studio .
