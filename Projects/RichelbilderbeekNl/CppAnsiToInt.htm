(C++) AnsiToInt

 

AnsiToInt is a convert code snippet to convert an AnsiString to integer.

 

///AnsiToInt converts an AnsiString to integer.
//From http://www.richelbilderbeek.nl/CppAnsiToInt.htm
int AnsiToInt(const AnsiString& s)
{
  return s.ToInt();
}
