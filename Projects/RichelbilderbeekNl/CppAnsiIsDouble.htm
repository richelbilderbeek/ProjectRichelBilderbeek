(C++) AnsiIsDouble

 

AnsiIsDouble is a check code snippet to see if an AnsiString can be converted to double.

 

 

 

 

 

Project and source code

 

Operating system: Windows XP

IDE: C++ Builder 6.0 Enterprise edition

Project type: Console Application

Compiler: BCC32.EXE version 6.0.10.157

Libraries used:

  • VCL: shipped with C++ Builder 6.0 Enterprise edition

 

 

 

 

 

main.cpp

 

//From http://www.richelbilderbeek.nl/CppAnsiIsDouble.htm
const bool AnsiIsDouble(const AnsiString& s, double& rDouble)
{
  return TryStrToFloat(s, rDouble);
}
