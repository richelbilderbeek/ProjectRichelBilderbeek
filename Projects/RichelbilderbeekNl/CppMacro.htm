(C++) Macro

 

Macros are preprocessor statements.

 

 

 

 

 

Advice

  • Avoid macros [1,8], except assert [2-6]
  • Avoid using preprocessor macros in header (.h) files except as include guards [7]
  • Use uppercase letters to name a macro [9,10]
  • Add a unique prefix to macro names [11]
