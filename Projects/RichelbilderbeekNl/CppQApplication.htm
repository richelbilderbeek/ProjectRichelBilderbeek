(C++) QApplication

 

QApplication is a Qt class that 'manages the GUI application's control flow and main settings' [1].

 

#include <QtGui/QApplication>

int main(int argc, char *argv[])
{
  QApplication a(argc, argv);
  Dialog w;
  w.show();
  return a.exec();
}
