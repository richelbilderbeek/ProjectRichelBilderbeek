(C++) TEMPLATE (qmake variable)

 

TEMPLATE is a qmake variable.

 

The value of TEMPLATE determines the kind of makefile that qmake will produce. TEMPLATE can have the following values [1]:

  • app: create a makefile for applications (default)
  • lib: create a makefile for building a library
  • subdirs: create a makefile for building targets in subdirectories. The subdirectories are specified with SUBDIRS
  • vcapp: create a makefile for building a Visual Studio application
  • vclib: create a makefile for building a Visual Studio library
