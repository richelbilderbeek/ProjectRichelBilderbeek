(C++) StrToAnsi

 

StrToAnsi is a conversion code snippet to convert a std::string to AnsiString.

 

#include <string>

//From http://www.richelbilderbeek.nl/CppStrToAnsi.htm
String StrToAnsi(const std::string& myString)
{
  return String(myString.c_str());
}
