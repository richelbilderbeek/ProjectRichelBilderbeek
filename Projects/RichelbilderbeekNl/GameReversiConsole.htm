(Game) Reversi Console

 

At the 30th March of 2008 I got an assignment to make a Console version of Reversi. Using and improving the same Reversi class as in my Reversi game with GUI of 2007 and using some of my code snippets I finished the assignment in 64 minutes. View the source code to see what is possible in 64 minutes.

 

Reversi Console is programmed in C++ using the IDE C++ Builder 6.0. It uses the Boost libraries.
