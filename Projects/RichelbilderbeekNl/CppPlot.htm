(C++) Plot

Examples

 

Only FOSS software. This list is incomplete and will remain incomplete.

 

  • QWT: C++ library
  • Maxima: Computer algebra system that allows easy plotting
  • Gnuplot: plotting environment, used as a workhorse by many programs
  • R: programming language that has build-in plotting
